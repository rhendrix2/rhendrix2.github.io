Vim Regex Reference

Commands

/ is while in normal mode will allow you to search for an expression.

s in command mode will allow you to replace expressions. This section applies mainly to the s command.

The format for the s command is:

:[range]s/[find]/[replace]/[options]

Ranges

Whole document:

:%s/.../.../g

Specific line range:

:23,46s/.../.../g

Current line to end:

:.,$s/.../.../g

g stands for global and means that all occurances in the range will be found. Leaving the g out will replace only the first occurance on each line.

Grouping

Groups can be capture with \(...\) and later referenced as \# where # is the number of the group.

\0 is the whole pattern.

Examples:

Replace foo with foobar:

.../foo/\0bar/g

Replace foobar with foofoo:

.../\(foo\)bar/\1foo/g

Special Characters

Unescaped

These characters have a special meaning and must be escaped to use them literally.

Character Meaning
\ Used to escape other characters
[] Used to define a character class
^ Line start or class negation
$ Line end
. Matches and character
* Matches 0 or more of what precedes it

Escaped

These characters have a special meaning, but have to be escaped to use it.

Character Meaning
+ Matches 1 or more of what precedes it
? Matches 0 or one of what preceds it
n Newline (only use when searching)
r Carriage return (use when replacing)
s whitespace character
S non-whitespace character
w "word" character (letters and numbers)
W non-word character
d digit
D non-digit
